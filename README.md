# Mini-AES
Mini-AES  
A miniature version or Rijindael, the Advanced Encryption Standard.

1. Compilation:  
  make

2. Encryption:  
  <code>encrypt(uint8_t* input, const uint8_t* key, uint8_t *output);</code>
  
3. Decryption:  
  <code>void decrypt(uint8_t* input, const uint8_t* key, uint8_t *output);</code>
